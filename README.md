# Auto-Recon Framework 

A powerful automation tool designed to streamline network reconnaissance and vulnerability discovery. This framework automates the execution of security scans and transforms raw technical data into actionable executive reports.

##  Key Features
- **Automated Scanning:** (Bash Module) Simplifies Nmap service discovery and script scanning.
- **Intelligent Parsing:** (Python Module) Extracts open ports and service versions from XML data.
- **Professional Reporting:** Generates Markdown-based summaries for security audits.

##  Tech Stack
- **Scripting:** Bash, Python 3.x
- **Tools:** Nmap
- **Output:** Markdown, XML
